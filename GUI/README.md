# RNAdetector GUI (Legacy)

> **Note**: This Electron desktop application has been superseded by the new React web client at `client/`. The GUI directory is retained for reference but is no longer actively maintained.

The new web-based client provides all functionality of the original Electron app plus:
- Server-side file browsing (no local filesystem dependency)
- JWT authentication with login/logout
- Dark mode futuristic UI
- Resource configuration (CPU/RAM) with recommendations
- Enhanced HTML reports with Materials & Methods

See `client/README.md` for the current frontend documentation.
